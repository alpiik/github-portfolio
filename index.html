About

I build web applications end to end, and I have a habit of not letting go until they actually work well. I'm an IT Systems Development student at TalTech in Tallinn, heading into my third year, where classroom fundamentals meet a steady stream of self-driven projects. As a developer I work across the full stack: React and Vue on the frontend, Python, Java, PHP with Laravel, Node.js, and SQL on the backend, with a soft spot for clean database design and code the next developer can read without wincing. I've also trained as a communications specialist with the Estonian Defence Forces, which taught me plenty about staying calm when systems misbehave.

A few things I'm proud of: a full blog platform built on Laravel and React, with secure authentication, public and private post sharing, and image uploads, deployed and live, not just sitting in a repo. A Texas Hold'em odds calculator powered by a Monte Carlo simulation engine I implemented twice, once in Java and once in JavaScript, partly to see how each language held up under the same problem. And a Flask app that scrapes product listings into SQLite and serves them through a filterable web UI, because checking prices by hand got old fast.

Away from the coursework, I'm usually on a football pitch or in the gym. I also have a background in game development, with a few ongoing projects in that space — it's where my software engineering habits get stress-tested by physics engines and impatient playtesters.
